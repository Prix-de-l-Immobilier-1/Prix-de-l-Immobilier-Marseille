
En termes démographiques, la ville de Marseille est la deuxième ville de France. Elle est de ces villes qui voient, depuis vingt ans, sa population augmenter d’année en année. Selon le [ site internet ] (https://www.marseille.fr/)  de la ville qui tient ses sources de l’INSEE, Marseille acquiert annuellement près de 3 244 habitants. Nous nous sommes intéressées ainsi au prix de l'immobilier à Marseille, les différences de prix selon les arrondissements et leur évolution. Pour ce faire, nous avons utilisé la base de données "Demande de valeurs foncières" (DVF) qui recense l'ensemble des ventes des biens fonciers de ces cinq dernières années en France. Nous avons également utilisé le site d'[ Euromed ] (https://www.euromediterranee.fr/) qui recense les divers aménagements Euromed dans l'ensemble de la ville et des jeux de données sur les autres aménagements marseillais.

# Comparaison de l'évolution du prix de vente immobilière entre le 2e et le 3e arrondissement en fonction des aménagements.

Afin de vérifier notre hypothèse, nous avons décidé de centrer cette étude sur les 2e et 3e arrondissements de Marseille. Nous avons choisi ces deux arrondissements car ils ont connu ces dernières années de nombreux aménagements, notamment avec les aménagements Euromed, considérables. 
Ce sont deux arrondissements de centre-ville qui présentent de nombreuses similitudes. En ce sens, une étude comparative nous a semblé bénéfique afin de répondre à notre hypothèse. Tout d'abord, voici un tour d'horizon du projet Euromed à Marseille.

## Le projet EUROMED

En 1995, Robert Vigouroux, maire de Marseille, lance le programme Euroméditerranée pour la mise en place d’un établissement public d’aménagement. Le programme Euroméditerranée acte 2, en 2007, relance et étend ce projet. Dans ce cadre-là, de nombreuses constructions et rénovations voient le jour, depuis 2014 notamment, permettant la redynamisation de certains quartiers. Nous avons établi une cartographie des constructions EUROMED entre 2014 et 2019 pour rendre compte de l'importance de ces aménagements. Comme nous pouvons le remarquer, ces aménagements concernent les 2e et 3e arrondissements de Marseille. Pour réaliser cette carte, nous avons extrait des données directement depuis le site Euromed.

# Conclusion 

Ainsi, notre étude s’est concentrée sur l’évolution des prix des ventes immobilières dans les 2e et 3e arrondissements de Marseille. Il s’agissait d’observer le lien de causalité entre la présence, la construction d’aménagements dans un quartier et les prix immobiliers. Notre étude a alors révélé que si la construction de nouveaux aménagements peut jouer un rôle dans le prix immobilier de certains arrondissements (ici le 2ème arrondissement), celle-ci n’entraîne pas une hausse systématique des prix. En effet, le 3e arrondissement a connu de nombreuses constructions entre 2014 et 2019 sans pour autant voir une augmentation générale des prix sur cette période.
La variation des prix de l’immobilier est ainsi le résultat de multiples facteurs parmi lesquels les aménagements semblent être une des composantes.


## La boîte noire

**1ère difficulté** : le volume des données nous a tout d’abord posé problème. En effet, il y avait différents jeux de données à trier, notamment pour les données DVF qui étaient séparées par arrondissements. Il nous a fallu les convertir sur [ WorkbenchData ] (https://workbenchdata.com/). Également, certaines lignes étaient identiques, ce qui nous a conduit à les dédupliquer sur le même outil.

**2ème difficulté** : le format des données DVF a posé problème afin de les intégrer à Google Sheets pour construire notre tableau croisé dynamique. En effet, les données étaient classées selon l’ordre suivant : « Année, Mois, Jour » non lu par l’outil. Ainsi, nous avons dû l’adapter manuellement.

**3ème difficulté** : l’outil [ Khartis ] (https://www.sciencespo.fr/cartographie/khartis/) nous a confronté à de nombreux problèmes. Tout d’abord, comme il n’existait pas de fond de carte de la ville de Marseille, nous avons voulu importer une carte que nous n’avons pas réussi à convertir au format « json ». Nous n’avons donc pas pu représenter une carte de chaleur sur l’outil. De plus, sur l’unique carte au format correct, celle présentant les aménagements Euromed, les données d’aménagements n’étaient pas enregistrées avec les adresses mais avec des données GPS que Khartis n’arrivait pas à relocaliser sur la carte.
